HERRAMIENTAS DE PROGRAMACION III
PAGINA DE GEOLOCALIZACION USANDO CCS, JS, HTML.

Se hace uso del API de Javascript “navigator” y leafleft para dibujar.
el mapa de Open Street Maps, de esta manera se visualizan (LATITUD Y LONGITUD) De la ubicacion en el mapa.
El uso de Live Server es clave para lanzar el servidor y probarlo desde otro dispositivo, asi mismo use la herramienta ngrok para una mejor ejecucion del servidor hacia otro dispositivo por medio del protocolo (HTTPS) ya que al solo conectarme por medio de LIVE SERVER la pagina no esta cifrada y puede ocasionar errores como la desactivacion de la ubicacion y no dejar activarla por seguridad, esto se mitiga con la solucion de conectar ngrok al LIVE SERVER para que la conexion sea cifrada y no de ningun error.
